# Common skills

This folder holds shared skills installed into each agent harness.
Keep files small and focused.

## Available
- `figma-api-extractor`: headless Figma REST extraction (frames + image URLs/downloads) for SSH/CI and MCP outages.
- `forge-swarm-orchestrator`: swarm spawn/ramp/monitor/stop workflow for forge + sv.
- `forge-workflow-operator`: author/validate/run/approve/resume workflow runs.
- `forge-team-task-ops`: create teams, manage members, drive task inbox lifecycle.
- `forge-node-ops`: route commands to mesh nodes and use node registry passthrough.
- `forge-mesh-ops`: provision/promote/demote mesh topology and health-check master routing.
- `linear-api`: run Linear GraphQL operations with Hem project secrets (default `project/digitech/linear`).
